# Site Web avec Déploiement Automatique
Ce site est déployé automatiquement à chaque modification grâce à GitHub Actions.
## Fonctionnalités
- Déploiement automatique sur GitHub Pages
- Site statique HTML/CSS/JS
## Comment ça marche ?
1. Modifiez le code
2. Commitez et poussez
3. GitHub Actions déploie automatiquement
